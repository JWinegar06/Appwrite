# 📋Task Manager

A modern task management application built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Appwrite**. The application features secure user authentication and full CRUD functionality for managing personal tasks, all wrapped in a Final Fantasy XV-inspired interface.

---

## ✨ Features

* 🔐 User Authentication

  * Create Account
  * Login
  * Logout

* 📝 Task Management

  * Create tasks
  * View tasks
  * Edit tasks
  * Mark tasks as completed
  * Delete tasks

* ⚡ Real-time Appwrite Database Integration

* 🎨 Final Fantasy XV-inspired UI

---

## 🛠 Technologies Used

* Next.js
* React
* TypeScript
* Tailwind CSS
* Appwrite
